# fast-api-tutorial

참고: https://fastapi.tiangolo.com/ko/

### 명령어
- 실행: `uvicorn app.main:app --reload`
    - main: main.py 파일
    - app: main.py 파일 내에 FastAPI()를 가지고 있는 객체
    - --reload: 코드가 변경된 후 서버 재시작 (개발 환경에서 사용)
